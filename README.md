
  # 💸MoneyKeep💸
  > **가계부 어플**


<br/>


## 📌 프로젝트 개요
```
평소 돈 관리를 잘 하지 못하고 돈이 항상 줄줄 세는 사람들에게 꼭 필요한 가계부를 어플로 개발해보았다. 
가계부 어플인 MONEY KEEP은 직관적으로 수입과 지출을 관리하고 통계를 조회할 수 있는 어플이다. 통계를
그래프로 한눈에 볼 수 있어 돈이 세는 구멍을 찾아 관리할 수 있다.
```
